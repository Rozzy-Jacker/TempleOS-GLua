simple gmod addon that trying to mimic TempleOS and it's Word gen.
there are basic random, basic word gen with vocab support and terminal entity
![20260322025852_1](https://github.com/user-attachments/assets/05c58109-bfca-4cf1-a57c-4ec88c695345)

convar holylua_enable_boot 0 for skip booting
convar holylua_terminal_view 1 for direct camera placement infront of terminal

[code]
SYSTEM COMMANDS:
help        - Show this help message
clear       - Clear the screen
shutdown    - Shutdown the system
reboot      - Reboot the system
version     - Show TempleOS version

DIVINE COMMANDS:
god         - Random Bible verse
godspeak    - Divine speech
godbits     - Random divine number (1-100)
randi32     - Random 32-bit integer

SOUND COMMANDS:
music       - Play TempleOS hymn
beep        - System beep
snd <file>  - Play custom sound
sndrst      - Stop all sounds

INFO COMMANDS:
memrep      - Memory report
date        - Current date
time        - Current time
cpurep      - CPU report
[/code]
